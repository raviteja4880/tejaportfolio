## 🛠️ Skills & Expertise

This portfolio highlights my proficiency across the full development stack:

- **Frontend**: HTML5, CSS3, JavaScript (ES6+), React.js
- **Backend**: Node.js, Express.js
- **Database**: MongoDB, Redis
- **Cloud & DevOps**: AWS (EC2, Lambda, CloudTrail), Git, Amplify
- **Design**: Responsive UI/UX, Glassmorphism, CSS Animations

---

## ✨ Key Features

*   🌟 **Premium UI**: Modern dark theme with glassmorphism and animated backgrounds.
*   📱 **Fully Responsive**: seamless experience across mobile, tablet, and desktop devices.
*   ⚡ **Dynamic Animations**: Smooth scroll reveal effects and interactive typing animations.
*   📧 **Contact Integration**: Fully functional contact form powered by Formspree.
*   📄 **Resource Center**: Integrated resume download and interactive certification display.

---

## 📁 Project Structure

*   `index.html`: Optimized semantic HTML with SEO meta tags.
*   `pstyles.css`: Advanced CSS using modern layout techniques and custom animations.
*   `pscript.js`: Lightweight JavaScript for interactive elements and scroll behaviors.
*   `assets/`: Repository for media, PDF resume, and certification documents.
